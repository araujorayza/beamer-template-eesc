\begin{frame}{Openloop Simulation}
Nonlinear error system and TS-fuzzy error system were subjected to the same input signal
\begin{equation}
u = \begin{bmatrix}
\sin(t)+\sin(1.3t)\\
-\cos(4t)-\cos(5.3t)\\
\sin(2t)+\sin(2.3t)\\
\cos(0.5t)+cos(0.3t)
\end{bmatrix}
\end{equation}
\end{frame}
